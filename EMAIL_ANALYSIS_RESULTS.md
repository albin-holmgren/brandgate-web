## 🎯 KEY INSIGHTS

### What Works:
✅ **firstname@company.com** - When it's the ACTUAL founder email
✅ **Verified personal emails** - Found on website/Instagram
✅ **Department emails** - For B2B (wholesale@, b2b@)
✅ **info@** - Only for very small companies (<30 employees)

### What Doesn't Work:
❌ **Guessing** patterns without verification
❌ **hello@** for small companies (often doesn't exist)
❌ **Assuming** firstname@ works for everyone

### Success Pattern Analysis:
- **short/firstname:** 14 successes (74%) - Works when verified!
- **first.last:** 2 successes (11%) - Sometimes works
- **generic:** 3 successes (16%) - Only for tiny companies

## 🚀 NEW WORKFLOW (No Guessing!)

### Step 1: Check Website
- Visit company.com/contact
- Look for ANY email listed
- If found → Use that email (don't guess!)

### Step 2: Check Instagram
- Visit instagram.com/company
- Check bio for email
- If found → Use that email

### Step 3: Check LinkedIn
- Founder profile → Contact info
- If email listed → Use it

### Step 4: Use Gmail to Verify Pattern (Last Resort)
- Test ONE pattern with Gmail
- Wait for bounce
- If bounce → Try different pattern (max 2 tries)
- If bounce again → SKIP

### Step 5: Send Only Verified Emails
- ✅ Email found on website/Instagram/LinkedIn
- ✅ Tested with Gmail, no bounce
- ❌ Never guessed without evidence
