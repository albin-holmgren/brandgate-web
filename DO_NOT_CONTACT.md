## How to Add to This List

### When a company says "Not Interested":
```
1. Add to this file immediately
2. Update LEADS_MASTER.md status to "Not Interested - DO NOT CONTACT"
3. Update Attio CRM status
4. Archive all emails
5. Do not add any other contacts from this company
```

### Format:
```markdown
| Company Name | 2026-03-03 | contact@company.com | "Not interested at this time" | DO NOT CONTACT |
```

## Verification Before Adding New Leads

Always check this list before adding any company to Attio:

```bash
# Search for company
grep -i "company name" DO_NOT_CONTACT.md

# If found: STOP - Do not add
grep -i "company name" LEADS_MASTER.md

# If found: STOP - Already in system
```
